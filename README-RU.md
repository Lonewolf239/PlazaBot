# Plaza Casino — Telegram-казино (Демо)

> **⚠️ ЮРИДИЧЕСКОЕ ПРЕДУПРЕЖДЕНИЕ:** Этот проект носит **исключительно образовательный характер** и **не предназначен для реального использования**.  
> Все ключи API и секреты удалены. Код демонстрирует архитектурные паттерны, асинхронное программирование и интеграцию с платёжными системами.  
> **Не используйте этот код для организации реального казино.** Законы об азартных играх различаются в разных странах; вы несёте полную ответственность за их соблюдение.

Plaza Casino — это многофункциональный Telegram-бот, имитирующий полноценное казино. Включает множество игр, реферальную систему с клон-ботами, криптовалютные платежи (через Crypto Pay), админ-панель и множество других продвинутых функций.

## Технологический стек

- **Язык:** Python 3.10+
- **Фреймворк:** aiogram (3.x)
- **Асинхронный сервер:** aiohttp (для вебхуков)
- **База данных:** aiosqlite (SQLite с асинхронной поддержкой)
- **Платежи:** Crypto Pay API (через aiocryptopay)
- **Графика:** Pillow, matplotlib
- **Внешние API:** CoinGecko (курсы валют), googletrans
- **Конфигурация:** python-dotenv

## Быстрый старт (только локальное тестирование)

### 1. Клонируйте репозиторий
```bash
git clone https://github.com/Lonewolf239/PlazaCasino.git
cd PlazaCasino
```

### 2. Создайте виртуальное окружение
```bash
python -m venv venv
source venv/bin/activate   # Linux/macOS
venv\Scripts\activate      # Windows
```

### 3. Установите зависимости
```bash
pip install -r requirements.txt
```

### 4. Настройте переменные окружения
Скопируйте `dotenv_template` в `.env` и укажите **тестовые** токены:
- `BOT_TOKEN` – от [@BotFather](https://t.me/botfather)
- `CRYPTOPAY_API_TOKEN` – от [@CryptoBot](https://t.me/CryptoBot) (используйте тестовую сеть, если доступно)
- `ADMIN_IDS` – ваш Telegram ID
- и т.д. (см. `dotenv_template` для полного списка)

> **Никогда не сохраняйте реальный `.env` в репозитории!** В репозитории только пример `dotenv_template`.

### 5. Запустите бота
```bash
python main.py
```
